# Customer Segmentation using K-Means Clustering

## Overview
This project performs customer segmentation for a retail store using the K-Means clustering algorithm. It groups customers based on their purchasing behavior.
## Features Used
- Annual Income
- Spending Score
## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
## How it Works
1. Load the dataset
2. Select relevant features
3. Apply K-Means clustering
4. Assign clusters to customers
5. Visualize clusters using a scatter plot
## Output
- Customers are grouped into different clusters
- Visualization shows different customer segments
- Helps identify high-value customers
## Key Learnings
- Unsupervised Machine Learning
- K-Means Clustering
- Data Visualization
- Customer Segmentation
